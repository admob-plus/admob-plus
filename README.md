[**AdMob Plus**](https://admob-plus.github.io) is the successor of [cordova-plugin-admob-free](https://github.com/ratson/cordova-plugin-admob-free) provides a clean API and is build with modern tools.

## Other features

- No Ad-Sharing
- No Remote Control
- Fully Open Source
- Well maintained
